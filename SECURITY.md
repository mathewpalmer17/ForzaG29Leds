# Security Policy

## Supported versions

Only the latest release is supported. Please update before reporting an issue.

## Reporting a vulnerability

This app reads UDP telemetry from Forza Horizon and writes raw HID output
reports to a connected wheel — it doesn't handle credentials, network
services, or remote input.

If you find a security issue (for example, a way to crash the app or send
unexpected HID data via a malformed telemetry packet), please report it
privately via [GitHub Security Advisories](https://github.com/mathewpalmer17/ForzaG29Leds/security/advisories/new)
rather than opening a public issue.
